2014/0914Sun衛藤美彩京都においでやす♡

こんばんみさみさ

 
 

 
 
 

今日は京都で 9thシングル『夏のFree&Easy』の個別握手会でした〜☆
 
 
 
久しぶりの京都♪
というより握手会自体がめっちゃ久しぶりで 変な緊張してました笑
 
 
 
でも今日の握手会は本当に楽しかったなぁ〜！
 
 
 
もちろんいつも楽しいんだけど、 ツアーの感想だったり 今度の舞台の話だったり 10thシングルの話だったり、色々な話をみなさんから直接聞けてとてもHAPPYな気分になりました♪
 
 
 
 
5部の最後では 終わったのが最後だったので一本締めをやらせて頂きましたよ(^o^)
 
 
 
と言っても、10thシングルについてちょっと話させてもらっただけで なんと肝心の一本締めをやり忘れました。笑
 
 
 
楽屋に帰って気付いたの〜σ(^_^;)
 
 
 
 
だからここで言わせて！
 
 
 
京都に来てくださったみなさん、ありがとうございました♡
 
 
 
よぉ〜おっポン！
この後も握手会が続いて行くので 来てくださる方はよろしくお願いします(^o^)♪
 
 
 
 
 
 
 
ゆったん♡
 
 
足はだいぶ治ったみたいで一安心...
良かった良かった( > < )
アンダーライブまでに完治してほしいなぁ。
 
 
 
 
 
 
 
服はこんな感じ♪
 
 
みさみさビームと あみあみビーム
 
 
今日はsnidelだった
 
 
ニット、肩のとこが鞄の重みに耐えきれず破けるという。
でもマネージャーさんの素早い対応で直して頂いた( > < )
  
 
 
 
4部だけこれ♪
 
 
ツアーTシャツ
 
千秋楽、明治神宮球場のだよ
 
 
 
 
 
 
今日もひめたんの腕を頂いたぞ
 
 
詳しくはひめたんブログへ♡
ひめたんの腕は美味しい(^o^)
 
 
 
 
 
 
寧々ころこと伊藤寧々ちゃんが卒業、二期生の矢田里沙子ちゃんと米徳京花ちゃんが活動辞退ということになりました。
 

まだ色々気持ちが整理できてないけど、何があってもそれぞれの選んだ道を応援するのが仲間だと思ってるんだ
 
 
きっとファンのみなさんも同じ気持ちですよね、、、(^ ^)？
 
 
だから今はまだ 残された時間をみんなで過ごしたい。
お別れはまだ言いません(^ ^)！
また別の記事で改めてちゃんと書きたいと思います！
 
 
 
 
 
 
 
 
そういえば次の京都個握は12月28日なんだよね(´･ω･`)♪
また年末！って言う方が多かったから、もうそんな季節かぁ〜としみじみ( > < )
 
 
 
 
ということはね。
京都もその前の12月23日の名古屋も
 
 
 
 
  
 
  
みさみさんた出没、むしろ、他のメンバーの乃木サンタ出没の可能性大
 
 
 
ですよ(^o^)/
サラッと言っておく。♡笑
 
 
 
 
 
 
 
さてさて次の記事は 最近公開された10thのMVのオフショット祭りします♪
 
 
ここで謎の写真を残しておく...(￣▽￣)
 
 
 
 
それではおやすみさみさ(^o^)♡♡
