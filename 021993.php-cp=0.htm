2014/1214Sun衛藤美彩ありがとう

  

 
 
昨日アンダーライブ。
すごかった〜〜（ ;  ; ）
 
 
 
みんなのパフォーマンスに感動した。
そして、自分がずっとアンダーだった時を思い出して、みんなと気持ちが重なっちゃった。
 
 
 
一曲一曲ごとに楽しかった思い出や悔しかったことや色んな気持ちが一気に蘇ってきて、イントロで涙がツーッと。
 
 
 
 
実は8thで自分が出たアンダーライブ以来の、初めてのライブ観戦。
今までもすごく行きたかったんだけど、どこかに観に行く事を躊躇している自分がいました。
 
 
 
 
自分でも気付いていた気持ちなんだけど、選抜に入るまでの7thまで一緒に過ごすことが多かったアンダーメンバー、
そのみんなが一致団結してて、どんどんいいものを作りあげていて、少し寂しくなっちゃってたのかなぁ...って
 
 
 
アンダーライブが全然人ごとに思えなくて 私も一緒に最後まで作り上げたかったなぁって気持ちがどこかにあったんだと思います。
 
 
 
でも今は選抜に選んでいただいているからここで頑張っていきたいし
 
 
 
本当に気持ちが矛盾してて そんな自分がすごく嫌で、こんな気持ちで観に行くのはみんなに失礼だなって思ってた。
 
 
 
でも、ファイナルだし"みんなのパフォーマンスを見届けるべきだ"って思ったの。
 
 
 
行って本当に正解でした。
本当、観に行ってよかった( > < )
全員のセンター本当良かった！
二期生や研究生にもスポットが当たっていて、みんなが輝いたライブだったと思います！
 
 
 
最近いろいろ考え込んじゃうことが多かったんだけどね、笑
みんなのライブのおかげで自分の気持ちと向き合えたから本当感謝の気持ちでいっぱい☆
本当にありがとう！
 
 
 
 
でも、私もみんなの頑張りに負けてられないから( > < )
感動をもらったから 私もいつか感動をお返ししたい。
 
 
 
これからもこんな素敵なみんなと一緒に、頑張っていきたいです。
 
 
 
 
8thアンダーライブの時の思い出の写真
まりかの生意気〜な顔。笑
 
 
 
 
 
そして今日から、ついにX'mas showがスタートしました♡
来てくださったみなさんありがとう！
一足早いクリスマスなのです！
 
 
 
今日のカップリングの順位はこれ！
 
 
３０位 遠回りの愛情  
２９位 扇風機
２８位 左胸の勇気
２７位 シャキイズム
２６位 孤独兄弟
２５位 生まれたままで
２４位 海流の島よ
２３位 そんなバカな・・・
２２位 水玉模様
２１位 やさしさとは


私が参加してるのは
 
扇風機
左胸の勇気
生まれたままで
そんなバカな・・・ の4曲♪
 
 
本当に、メンバーも順位は知らされていないから 明日はどの曲がランクインするのかドキドキです(o_o)
 
 
 
明日は二公演がんばるぞー！
今日のライブで私、生駒ちゃんの存在の大きさを感じたから...明日は生駒ちゃんの分まで頑張って楽しみたいと思います！
 
 
 
 
 
 
明日頑張れるように、みんなのパワーもちょうだい( > < )☆
