夏バテ気味で食欲が出ない...
みさみさです。
  
  
  
みんなは夏バテしてないですか？
食欲少ないんだけど
フルーツならいっぱい食べれる♪
ミカンにパイナップルにキウイ
  
  
あと、ミックスジュース？
野菜とかフルーツとか
混ぜたジュースも
自分で作ってみようかなぁ～。
  
何と何を混ぜれば 
おいしいのできるよっって
知ってる人いる～？？
居たらレシピ教えて♪(´ε｀ )
  
あ！！！でもねご飯もちゃんと
食べてるからね！！笑
  
  
そして、なでしこジャパン
決勝進出おめでとうございます
ハーフタイムまで 頑張って
起きてたけど気付いたら朝....。
最後まで見たかった （ ;  ; ）
  
今は 男子サッカーを観てます。
夜更かし～～～だけど
どうしてもリアルタイムで観たい。
  
※ちなみにみさの家は
テレビの録画機能がない。笑
買おうっていつも思ってるけど
優先順位が低いのです(｡-_-｡)ぁ
  
みさの注目選手は清武選手。
同じ大分県出身の方なので
特に期待しています(*^o^*)
  
  
  
そして明日は 甲子園開幕☆
甲子園が始まると
あ～夏がやってきたなぁと
実感したりします！！笑
  
大分に居た頃は
予選大会の時期になると
放課後 親とか友達と一緒に
新大分球場まで観に行ってたな～。
懐かしい(*^^*)
  
  
大分県からは
杵築高等学校が出場します。
本当に頑張ってほしい(/ _ ; )！
応援してます！！！
  
  
今年はどんなドラマが
生まれるのかなぁ。
  
  
今日はスポーツのこと
いっぱい書きました♪
  
あ～なんかみさも久しぶりに
バレーしたくなっちゃったー！
元バレー部同士、
さゆりんご誘って
バレーしてこようかなっ♪(´ε｀ )
  
  
  
  
  
明日もレッスンがんばろ！
じゃ、またね♪
  
にゃにゃせまるっっに
今日相談にのってもらったぁ！
ありがとだいすき(>_<)
  
  
  
  
あ、もちろんだけどね、
みんなのことも大好きです。
