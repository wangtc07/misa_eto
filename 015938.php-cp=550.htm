2013/1230Mon衛藤美彩2013年最後の握手会 ！and生駒へ！ misa♡

こんばんみさみさ！
  
  
二日間の握手会 来てくださったみなさんありがとうございました♪
  
  
まず土曜日の全国握手会。
これでバレッタの全握が全て終了しましたね（ ;  ; ）
  
  
あっという間だな〜〜早かったな〜〜と余韻に浸りつつ、握手会。
  
  
ペアはあしゅりん♡
  
  
 
  
ツーショットが全てブレてた...
よってみさ先輩自作の加工写真...
 
 
 
みなさん本当ありがとうございました！
とても楽しかったです！
京都の方って本当、のんびり優しい方が多いなぁという印象でした。
 
  
 
(ホテルはゆったんと相部屋でした♡)
 
  
 
 
  
 
そして今日は個別握手会♪
  
  
ブログで書いていた通り お約束のボブヘアにしました
  
 
  
  
一部と二部はロングで 三部がボブ
  
  
  
休憩の間に髪切ったの！って言ってみなさんを驚かしました。
その反応が面白くて何回かしちゃいました！
ごめんね( ›◡ु‹ ) ♡
  
 
好評で嬉しかったです！
だから衛藤はあることを決心しました
(何かはまだ内緒)
  
また今度のブログで♪
  
  
  
全身は撮り忘れたんですがグレーのパーカーワンピースでめちゃめちゃラフな格好でした(*^_^*)
  
 
  
 
前回より本当に多くの方が私のレーンに並んでくださってて会いに来てくださって。
  
  
次のシングルもお久しぶりって話せたらいいな(*^_^*)
2013年最後の握手会！
本当にありがとうございました！
  
  
 
まいまいもラフな格好だったー♪
 
 
  
 
  
  
最後に 12月29日は生駒ちゃんの18歳の誕生日♡
 
  
みんなでお祝いしましたー！
握手のレーンに入る前 生駒ちゃんのファンのみなさんにもおめでとうって言ったら笑われちゃいました...笑
  
  
いこたんとは ブログではあまり書いていないけど実はすごく仲良しです。
 
  
基本、朝会ったらおはようの代わりにいこたんの方から胸やお尻をソフトタッチしてきます。
  
(みさもそれに何事も無かったようにおはよ〜と言います)
もう二年もこのスタイルなので慣れました！笑
 
  
  
ふざけてるだけじゃなくて真面目な話もよくします。
  
私はこう思うんだけどみさ先輩はどう思う？って意見を求めてきてくれたり。
信頼してくれてる感じが本当に嬉しくて またそんな生駒が可愛くて可愛くて...
 
だから私も色んなことを生駒に相談します(*^^*)
  
 
  
私はいつも後ろの方から一番前に立つ生駒の背中を見てきました。
色んなプレッシャーをその小さい背中で受け止めている姿を見てきて、私には何ができるかな？と考えていました。
  
  
でも...もうそんな心配はいらないくらい 今の生駒はたくましくてすごく強く成長したと思います。
 
私は一体誰目線なんだ、、、。笑
  
  
私もその勢いに負けてられないっ！笑
 
 
いつも先頭に立って乃木坂の事を考えてくれて、引っ張ってくれて本当に誰よりもしっかり者。
でも、生駒は生駒のペースで大丈夫だよ(*^^*)
  
  
これからも一緒に頑張っていこうね！
魅力たっぷりのいこたん、大好き！！
  
  
 
  
  
と熱く語ってしまいましたが、みなさん！
今年も残りあと2日ですよ！！
どうしよう！！
  
  
2013年が終わってしまう〜。
  
  
  
今年は本当にたくさんのことをやらせていただいて、一年をゆっくりじっくり振り返りたいんですけど 12月も嵐のように過ぎ去っていきました...
 
  
でもちゃんと振り返らなきゃ。
思い出さなきゃいけない、いろんな感情が眠ってるはず。
  
  
  
今年一年間のこと、2014年の抱負、ちゃんと気持ちを整理してまたブログを更新します(*^^*)
  
  
みなさんも 忙しいと思うけど 一緒に今年一年間を振り返ってみましょう♪
  
  
 
 
 
それでは、おやすみさみさ。
  
  
 
 
あ！
エア握手！
  
 
  
忘れるところだった〜
危ないっ笑
  
  
 
ゆっくり休んでね(*^^*)Zzz

会長～

こんばんみさみさ＼(^o^)／
では前回の続きを書きます。m(_ _)m
( 見てないなら、読んでな♡)

初握手を期に、みさに興味を持ち始めました。
いったい、どんなコなんだろう？(///ω///)
ブログやネット、色々調べていくうちに、どっぷり、みさ♡にハマっていきました…(照れ)
乃木坂握手会に参加する前は、みるきーしか♡とか言ってた私が、気づいたら口に出すのは、みさ♡ばっか（笑） 友達からは、「みさに推し変したでしょ？」と、からかわれる始末…。自分でもビックリするくらい、気持ちが揺らいでました。(^_^;)

まあ…みるきーと初握手したら、ハッキリするでしょう！
そう思って迎えた、みるきーとの初握手。
たしかに可愛い♡神対応♡握手会の最中は終始デレデレ♡ε=(///ω///)=33
これが、みるきー♡なんだと実感、大満足な、みるきーとの初握手会となりました。

しかし…
みさへの気持ちに変化はありませんでした。
日に日に、みさへの気持ちが大きくなり、みるきーへの想いが小さくなっていく。
周りから見たら完全に推し変してるような状態。
なのに、頑なに推し変したと認めない自分が居ました…
推しメンは、あくまで、みるきー。みさは好き♡
そんな苦しい言い訳をする日々が続きました。

ハッキリしない状態が続いてる間に、4th、5thシングルと、みさは選抜に入れませんでした。
なんで選抜に入れないのだろう？
可愛いし、歌もダンスも、MCだって出来る万能タイプなのになぜ？

どっちを推しているのか？ハッキリしないモヤモヤと、みさが選抜入りできないイライラが入り混じって迎えた、乃木坂5thシングル京都全国握手会。
この日は、握手会後に6thシングルの選抜メンバーが発表されるということが事前に予告されてました。
次こそは…
みさの初選抜入りが発表された時、その現場にいて一緒に喜びを分かち合いたい…そう思って会場入りしました。

結果は選抜入りならず…

翌日の個別握手会での、みさの悲しそうな顔…ファンに申し訳なさそうな顔が今でも忘れられません…

帰りの車の中で、みさが選抜入り出来なかった理由を考えてました。そして気付いちゃいました…

…私のせいだ。みさと、みるきー。どっちを推してるのかハッキリしないから…中途半端な推し方をしてるから、みさは選抜入りを逃したんだ…（泣）

その時から、私は、みるきーから、みさへの推し変を認め、みさみさ単推しを決意しました。
私なりに、これが推し変した理由だと考えてます。。(^_^;)


余談ですが7thシングルで、みさは初の選抜入り。
私の推し変は関係ないと、わかっておりますが…
推し変して良かった♡みさ推しになって、ホント良かったなと思ってます。
長くなり、訳が分からなかったらゴメンなさい。
m(_ _)m
