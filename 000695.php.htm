こんばんはー！
みさみさですよ〜ん
　
　
みんなからのコメント
嬉しく読んでます(>_<)。

　
　
ありがとう！！

　
　
　
　
　
　
今日はね
凄く嬉しい出来事がありました。
　
　
　
　
朝の電車での出来事。
　
　
　
　
電車を降りるときに
Suicaを車内に
落としてしまって(x_x;)うぅ
　
　
　
　
でもみさはそれに気付かず↓
急いでいたのもあり
スタスタかなりの速歩きで
乗り継ぎのために
違うホームまで歩いていたんです。泣
　
　
　
　
そしたら電車内で
カードを拾ってくれた方が
みさの後を追いかけて
届けに来てくださったんです！
　
　
　
　
。・゜・(ノД`)・゜・。
　
　
　
　
落として気付かない
みさって本当、、。あぁ。。
°･(ノД`)･°･.°‥

　
　
　
　
朝の通勤ラッシュで
ただでさえ混んでいる中
追いかけて届けてくれたことに
その方の優しさに
感動しました(/_;)。。。。。

ありがとうございます（ ; ; ）

　

　
　
でも、、、、、
もうちょっとしっかりしろ自分！
(´･ω･｀)

　
　
もうこれからは気をつけよ(/ _ ; )
迷惑かかるしね(/ _ ; )
　
　

そしてみさもそんな
素敵な人になりたいな！！！！
　
　
　
　
みんなも、くれぐれも
落し物には気をつけてね！！！！！
　
　
みさが言っても説得力ないか（ ; ; ）笑
でもでもっ、注意してね。
　
　

　　
　

では明日も頑張ろーう！！！
　
　

おやすみさみさ〜(*^_^*)笑
